## Installation
* Run Mock API (Run following in mock-api folder)
```
npm install
npm run start
```	

* Install dependencies (Run following in magnitt-test folder)
```
npm install
```

* Start app in dev mode
```
npm run dev
```

* Start app for production
```
npm run build

npm run start
```
